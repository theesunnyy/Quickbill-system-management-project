# 🧾 QuickBill – Smart Billing & Expense Manager

QuickBill is a Java-based desktop application designed for local shop owners to manage billing, expenses, inventory, and reports efficiently. The system provides a user-friendly interface with powerful features like invoice generation, expense tracking, and data backup.

---

## 🚀 Features

- 🧾 Generate and print bills (invoices)
- 💰 Track daily expenses and income
- 📦 Manage items and inventory
- 👤 User authentication (Login system)
- 📊 View reports and analytics
- 🔄 Backup & Restore data
- 📷 QR Code display for payments
- 📄 Receipt preview before printing

---

## 🏗️ Project Architecture

This project follows **MVC (Model-View-Controller)** architecture:

- **Model** → Data classes (Bill, Item, User, Expense)
- **View** → Swing UI (Dashboard, Panels, Forms)
- **Controller/DAO** → Business logic & database operations

---

## 🛠️ Technologies Used

- **Language:** Java  
- **UI Framework:** Java Swing  
- **Database:** MySQL  
- **Build Tool:** Maven  
- **Libraries Used:**
  - MySQL Connector
  - FlatLaf (UI Design)
  - JavaMail API
